# UI 组件库简介

本组件库基于 React，适用于中后台、管理系统等场景，支持主题定制与现代化开发体验。组件风格参考 [shadcn/ui](https://ui.shadcn.com/) 并结合团队实际需求进行优化。

---

## 快速开始

**安装依赖**

```bash
pnpm add @repo/ui
```

**引入全局样式**

在项目入口文件（如 `main.tsx` 或 `app.tsx`）中引入：

```ts
import '@repo/ui/styles/globals.css';
```

**使用组件**

```tsx
import { Button } from '@repo/ui/components/button';

<Button variant="primary">按钮</Button>
```

---

## 组件一览

- **Button 按钮**：多变体、多尺寸，支持图标与加载态
- **Avatar 头像**：用户头像展示
- **DropdownMenu 下拉菜单**：多级菜单与交互
- **Sonner 通知**：轻量级消息提示
- **Label 标签**、**Input 输入框**、**Form 表单**、**Card 卡片** 等

详见左侧各组件文档。
